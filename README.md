# 🍽️ RTE_STORE

A Restaurant Inventory & Stock Management System built with Python and SQLite. This project helps restaurants manage inventory, track stock movements, record purchases, and monitor available stock efficiently.

## 🚀 Features

- 📦 Inventory Management
- 🏪 Store & Location Management
- 📊 Stock Ledger
- ➕ Purchase Entry
- ➖ Stock Issue & Consumption
- 🔍 Search Inventory Items
- 💾 SQLite Database
- 📈 Real-time Stock Tracking

## 🛠️ Tech Stack

- Python
- SQLite
- VS Code


## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/sumit800156-wq/RTE_STORE.git
```

2. Go to the project folder

```bash
cd RTE_STORE
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the project

```bash
python main.py
```
